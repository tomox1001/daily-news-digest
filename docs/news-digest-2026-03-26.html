AnthropicがClaude Code「自動モード」をリリース — 承認なし自律実行で開発者コミュニティが大反響

Anthropicは3月24日、Claude Code Auto Modeを研究プレビューとしてリリースした。従来のClaude Codeはファイル書き込みやbashコマンドの実行ごとにユーザー承認を要求していたが、新モードではAIベースの安全性分類器が各アクションを自動評価し、安全なものは承認なしに実行する。大量ファイル削除やデータ漏洩パターンなど危険な操作は自動ブロックされる仕組みで、「完全承認スキップ（dangerously-skip-permissions）」と「都度承認」の中間的安全性を実現。Teamプランのユーザーを対象にEnterpriseおよびAPIへの展開を順次予定している。TechCrunch、Simon Willison、rohanpaul_aiなど主要テックアカウントが一斉に拡散し、X上では開発者の「承認疲れ（approval fatigue）」を解消する公式回答として高評価を得た。
